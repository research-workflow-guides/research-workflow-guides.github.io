VS Code Installation

Install Visual Studio Code, add LaTeX Workshop, and confirm the editor is ready for the first project.

Install VS Code

1. Download VS Code

Use the official download page and install the normal desktop version for your operating system.

2. Open VS Code once

Finish the installer, launch VS Code, and confirm the editor opens normally.

Official VS Code downloads →

Install LaTeX Workshop

1. Open the Extensions panel

Use the Extensions icon in the left sidebar or the Command Palette.

2. Search for LaTeX Workshop

Install the extension published by James Yu.

3. Wait until the extension is fully loaded

After installation, keep VS Code open for a moment so commands and side features can register.

LaTeX Workshop on the VS Code Marketplace →

Self-check

Before you move on

Confirm that VS Code opens, the extension is installed, and LaTeX Workshop commands appear in the Command Palette.
