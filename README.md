# Research Workflow Guidebook

In mathematics research, the usage of LaTeX is inevitable, making it an essential tool for mathematicians throughout their careers. This seminar introduces Visual Studio Code, a powerful editor that has gained significant attention recently for its versatility and adaptability. 
The session will guide you on how to utilize LaTeX within Visual Studio Code. 
The seminar also offers a tutorial of Git, a handy tool for version control and team-based projects, thereby promoting efficient workflows and collaboration with peers. 
Lastly, the seminar shares four powerful tools – GitHub Copilot, JabRef, Mathpix, and ChatGPT – each carefully selected to streamline your work processes.
