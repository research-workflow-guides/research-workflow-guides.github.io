## VS Code에서 로그인

VS Code의 Accounts 메뉴나 Source Control 관련 안내에서 GitHub 로그인 흐름을 시작합니다.

## 브라우저 인증 완료

보통 VS Code가 브라우저 인증 페이지를 엽니다. 이전 단계에서 준비한 같은 GitHub 계정으로 인증을 마칩니다.

## 연결 확인

VS Code로 돌아와 연결된 계정이 올바르게 보이는지 확인합니다. 이후 repository 관련 동작이 GitHub 기준으로 이어져야 합니다.

## 점검

VS Code에 올바른 GitHub 계정이 보이고 로그인 창이 계속 반복되지 않으면 저장소 생성 단계로 넘어갑니다.
