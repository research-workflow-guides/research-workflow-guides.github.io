1.1 LaTeX 설치(Installation)

3가지 선택지가 있습니다. 필요에 맞게 선택하여서 설치를 진행하세요.

There are three options available. Choose according to your needs and proceed with the installation.

Option 1

TeX Live (fast)

10분만에 비교적 쉽게 설치 가능, 나중에 패키지를 일일이 다운 받아야 함.

Quick and easy installation in about 10 minutes, but you'll need to manually download packages later.

자세히 보기 →

Option 2

TeX Live (slow)

엄청 쉽지만 오랜 설치 시간, 나중에 패키지를 다운 받을 필요 없음.

Easier installation but takes longer, and you won't need to download packages later.

자세히 보기 →

Option 3

MiKTeX (Windows only)

10분만에 비교적 쉽게 설치 가능, 나중에 패키지를 자동으로 다운 받을 수 있음. Windows만 지원함. Mac에서도 지원하는지 알 수 없음.

Can be installed relatively easily in about 10 minutes, and it can automatically download packages later. Only supported on Windows. Not sure for Mac.

자세히 보기 →

Note

본 강의에서는 MiKTeX을 사용합니다

VS Code의 LaTeX-Workshop은 공식적으로 MiKTeX보다는 TeX Live를 사용하는 것을 강력히 추천합니다. 그러나 본 강의에서는 MiKTeX을 사용합니다. 보다 자세한 내용은 공식 사이트를 참고하세요.

VS Code's LaTeX-Workshop strongly recommends using TeX Live over MiKTeX officially. However, in this tutorial, we will be using MiKTeX. For more detailed information, please refer to the official website.

James-Yu/LaTeX-Workshop — official repository
