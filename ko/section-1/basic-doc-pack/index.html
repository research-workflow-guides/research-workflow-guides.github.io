파일 설명

Seed document/

문서 작업을 시작하는 기본적인 파일들이 담겨져 있는 폴더입니다. Seed document.tex를 컴파일하려면 bookmark, enumitem 패키지를 설치해야 합니다. Section 1.1의 Step 9, 10을 참고하세요.

Check document.tex

이 파일이 오류 없이 컴파일된다면 설치가 완료된 것입니다. 컴파일 방법은 Section 2.1에서 배웁니다.

keybindings.json

단축키 설정 파일입니다. Section 2.1.1에서 다루어집니다.

latex.json

기본적인 User Snippets 파일입니다. User Snippets는 Section 2.1.5에서 배웁니다.

settings.json

VS Code 설정 파일입니다. Section 2.1.1에서 다루어집니다.
