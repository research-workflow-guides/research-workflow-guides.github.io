1.3 Git 설치

버전 관리 및 GitHub 협업을 위한 Git을 설치합니다.

Step 1

오른쪽 배너를 클릭해서 홈페이지로 이동하세요.

Git 다운로드 →

Step 2

사용하는 컴퓨터의 OS에 맞게 Git을 다운 받으세요.

Step 3

설치를 진행하세요. 첫번째 그림의 화면에서 시작합니다.

가운데 그림에서처럼 "Use Visual Studio Code as Git's default editor"를 선택해주세요. 안하셔도 큰 문제는 없습니다. 나머지는 변경하지 않고 진행해주세요.

설치가 완료되면 마지막 그림의 화면을 확인할 수 있습니다.
